# Dossier Images

Placez votre photo professionnelle dans ce dossier.

## Instructions

1. Nommez votre photo : `photo.jpg` (ou `photo.png`)
2. Formats acceptés : JPG, PNG, WebP
3. Taille recommandée : 
   - Minimum : 800x800 pixels
   - Optimal : 1200x1200 pixels ou plus
   - Format : Carré (1:1) pour un meilleur rendu

## Note

Si votre photo a un nom différent, modifiez le chemin dans `index.html` :
- Ligne ~73 : `<img src="images/votre-photo.jpg" ...>`
- Ligne ~121 : `<img src="images/votre-photo.jpg" ...>`
